# 아이템 55. 옵셔널 반환은 신중히 하라

## 자바 8 이전

자바 8 이전, 메서드가 특정 조건에서 값을 반환할 수 없을 때 취할 수 있는 선택지

- 예외를 던지기
- (반환 타입이 객체 참조라면) null을 반환하기

두 방식 모두 허점이 존재한다.

- 예외를 던지기
  - 예외는 진짜 예외적인 상황에서만 사용해야 함
  - 예외를 생성할 때 스택 추적 전체를 캡쳐하므로 비용도 만만치 않음
- (반환 타입이 객체 참조라면) null을 반환하기
  - 별도의 null 처리 코드를 추가해야 함
  - 언젠가 NPE 발생할 수 있음
